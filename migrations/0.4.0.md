# Migration to v0.4.0

## ratchets introduced

The devcontainer now installs the ratchet tool, which holds counts of undesirable
code patterns non-increasing, and projects gain a ratchet config. (Renamed to
`ratchets` / `ratchets.toml` in v0.6.0 — see the later guides.)

### Steps
1. Rebuild your devcontainer so the ratchet binary is on PATH.
2. Run the ratchet check and baseline budgets if it reports counts.
