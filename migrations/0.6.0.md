# Migration to v0.6.0

## ratchet.toml renamed to ratchets.toml

The configuration file `ratchet.toml` has been renamed to `ratchets.toml`.

### Steps
1. If you have a `ratchet.toml` file, rename it: `mv ratchet.toml ratchets.toml`
2. Update any CI scripts or documentation that reference `ratchet.toml`

## beads migrated to beads_rust

The bead tracking system has migrated from the Go-based `bd` tool to the Rust-based `br` tool.

### Steps
1. Install beads_rust: `cargo install --git https://github.com/Dicklesworthstone/beads_rust.git`
2. Replace `bd` with `br` in all commands
3. If you have an existing `.beads/issues.jsonl`, run `br sync --import-only`
4. Verify with `br list`
