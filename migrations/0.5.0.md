# Migration to v0.5.0

## WORKFLOW*.md replaced by .claude/agents/*.md

Agent roles moved out of `WORKFLOW*.md` into per-role files under
`.claude/agents/` (coordinator, coding, judge, tidy, reflection). `apply` writes
the new files but does not delete the old ones.

## CLAUDE.md is now a symlink to AGENTS.md

### Steps
1. Delete your old workflow files: `rm WORKFLOW*.md`.
2. If a regular `CLAUDE.md` file exists, remove it so `apply` can create the
   `CLAUDE.md -> AGENTS.md` symlink: `rm CLAUDE.md && rust-bucket apply`.
