# Migration to v0.9.0

## ratchets.toml and STYLE_GUIDE.md are now seed files

`apply` writes these only if absent and never overwrites them again — your
project owns them. Edit them freely; future upgrades will not clobber your
changes. Projects missing `ratchets.toml` are seeded with one.

## ratchets binary install documented

Ensure `ratchets` is on PATH: `cargo install ratchets` (pin the version your
devcontainer pins). See https://crates.io/crates/ratchets.

### Steps
1. Existing `ratchets.toml` / `STYLE_GUIDE.md` are left as-is — nothing to do.
2. Confirm the tool is present: `ratchets --version`.
